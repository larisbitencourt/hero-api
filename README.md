# Hero API 🦸‍♂️

API progressiva desenvolvida com **NestJS** e **TypeScript**, focada em escalabilidade e integração com serviços cloud. Este projeto faz parte do meu portfólio de transição para o desenvolvimento de software, aplicando conceitos de **Clean Architecture**, **SOLID** e **TDD**.

## 🚀 Funcionalidades Principais
- **Gestão de Heróis:** CRUD completo com persistência em banco de dados PostgreSQL.
- **Integração AWS:** Gerenciamento de uploads de imagens/arquivos utilizando **AWS S3**.
- **Observabilidade:** Módulo dedicado para logs de sistema e monitoramento de erros.
- **Webhooks:** Suporte para integração de eventos em tempo real com serviços externos.
- **Processamento de Dados:** Script utilitário para manipulação e exportação de arquivos CSV.
- **Banco de Dados:** Controle de versão de schema via **Migrations** com TypeORM.

## 🛠 Tecnologias e Ferramentas
- **Backend:** NestJS, Node.js, TypeScript.
- **Cloud:** AWS (S3 / Lambda).
- **ORM:** TypeORM (PostgreSQL).
- **Qualidade:** Jest (Unitários e Integração), ESLint, Prettier.
